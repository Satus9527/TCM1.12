## 📝 如何修改环境变量

### 后端 .env

**方法1: 记事本**
```bash
notepad .env
```

**方法2: VS Code**
```bash
code .env
```

**方法3: PowerShell查看**
```powershell
type .env
```

---

### 前端 .env.development

**方法1: 记事本**
```bash
cd czb_test4
notepad .env.development
```

**方法2: VS Code**
```bash
cd czb_test4
code .env.development
```

**方法3: PowerShell修改**
```powershell
cd czb_test4
echo VITE_API_BASE_URL=http://localhost:3000/api > .env.development
```

## 🔄 重要提示

### 修改后需要重启 ⚠️

**后端**:
1. 修改 `.env` 文件
2. 停止服务器 (Ctrl+C)
3. 重新运行 `启动服务器.bat`

**前端**:
1. 修改 `.env.development` 文件
2. 停止开发服务器 (Ctrl+C)
3. 重新运行 `npm run dev`
