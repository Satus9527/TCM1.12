### 对于AI团队

2. **获取** AI团队URL
3. **执行** 测试用例
4. **反馈** 测试结果

## ⚠️ 重要提醒

### 格式规范至关重要

为了稳定集成，AI团队**必须**返回以下格式：

**推荐服务**:
```
辨证为：[证型]。方剂ID：[uuid]。
```

**分析服务**:
```
<JSON_START>{"overall_properties": {...}, "functions_analysis": {...}}<JSON_END>
```

### URL更新机制

由于Colab使用临时URL，AI团队需要：
- 每次重启后及时通知最新URL
- 建立稳定的通知机制（邮件/Slack/钉钉等）
